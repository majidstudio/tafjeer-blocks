# 🚀 تفجير البلوكات! — Deploy to Vercel in 10 Minutes

## You need: A computer with internet. That's it.

---

## OPTION A: Fastest (GitHub + Vercel) — 10 minutes

### Step 1: Create GitHub account (skip if you have one)
1. Go to **github.com**
2. Click "Sign Up"
3. Use your email, create username and password
4. Verify email

### Step 2: Create a new repository
1. Click the **"+"** icon (top right) → "New repository"
2. Name: **tafjeer-blocks**
3. Select **Public**
4. Check ✅ "Add a README file"
5. Click **"Create repository"**

### Step 3: Upload the game files
1. In your new repository, click **"Add file"** → **"Upload files"**
2. Drag and drop ALL files from the `tafjeer-web` folder:
   - `index.html`
   - `package.json`
   - `vite.config.js`
   - `src/main.jsx`
   - `src/App.jsx`
3. **IMPORTANT:** Keep the folder structure! The `src` folder must contain `main.jsx` and `App.jsx`
4. Click **"Commit changes"**

### Step 4: Deploy on Vercel
1. Go to **vercel.com**
2. Click **"Sign Up"** → **"Continue with GitHub"**
3. Authorize Vercel to access your GitHub
4. Click **"Add New..."** → **"Project"**
5. Find **tafjeer-blocks** → click **"Import"**
6. Framework Preset: should auto-detect **Vite**
7. Click **"Deploy"**
8. Wait 1-2 minutes... ⏳
9. 🎉 **DONE!** You get a URL like: `tafjeer-blocks.vercel.app`

### Step 5: Share it!
Copy your URL and send it everywhere:
- WhatsApp groups
- TikTok bio
- Twitter

## OPTION B: Even Simpler (StackBlitz) — 5 minutes

If GitHub feels complicated:

1. Go to **stackblitz.com**
2. Click "Start a new project" → "React (Vite)"
3. Delete everything in `src/App.jsx`
4. Paste the ENTIRE content of the game's `App.jsx` file
5. Delete everything in `src/main.jsx`
6. Paste the content of `main.jsx`
7. Replace the content of `index.html` with our `index.html`
8. The game runs instantly in the browser!
9. Click "Share" to get a URL

## After Deploying — Test Checklist

Open your URL on your phone and check:
- [ ] Game loads (you see تفجير البلوكات! title)
- [ ] You can drag pieces onto the board
- [ ] Rows/columns clear when completed
- [ ] كفووو text appears
- [ ] Sounds play (tap screen first to enable audio)
- [ ] Score saves when you close and reopen
- [ ] "استمر" button appears when you return to a saved game
- [ ] Share button works (sends WhatsApp/other apps)
- [ ] Game works on both iPhone and Android
